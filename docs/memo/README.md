## デーモン (daemon)とは

* メモリ上で「さぁ、いつでもかかってこい」と待機している常駐プログラムのUNIX系OSにおける呼び名

* デーモンさんの名前には、最後に「d」を付けるのが慣例になっています。
httpdやftpd、syslogdなど、後ろに「d」が付いていたら「あっ！こいつはデーモンさんだな！」と判断して、かまいません
